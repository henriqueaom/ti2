Quem somos?

Quem somos? Somos um grupo de Estudantes da PUC Minas derivado da matéria Trabalho Interdisciplinar 2, formado por 5 membros.

O projeto desenvolvido consiste em um sistema de estacionamento automatizado, projetado para simplificar a experiência dos usuários e otimizar a gestão de vagas e pagamentos. O software utiliza uma API avançada para leitura automática das placas dos veículos, eliminando a necessidade de interação manual durante o processo de entrada e saída do estacionamento.

Funcionamento

Leitura de Placas:
Ao entrar no estacionamento, as câmeras instaladas capturam a imagem da placa do veículo. Essas imagens são processadas pela API, que extrai e identifica os caracteres da placa, registrando automaticamente a entrada do veículo no sistema.

Registro e Monitoramento:
O sistema armazena os dados da placa, juntamente com a hora de entrada, em um banco de dados seguro. Esse registro permite o monitoramento em tempo real de todos os veículos presentes no estacionamento, oferecendo uma visão detalhada para os administradores.

Pagamentos Automatizados:
Quando o usuário decide deixar o estacionamento, a mesma API realiza uma nova leitura da placa para registrar a hora de saída. Com base no tempo total de permanência, o sistema calcula automaticamente o valor a ser pago. O pagamento é realizado de forma totalmente automatizada, vinculando a placa do veículo aos dados de pagamento previamente cadastrados pelo usuário. Isso pode ser feito por meio de métodos como cartão de crédito ou débito, associados à placa do veículo no sistema.

Benefícios do Sistema:
Eficiência: Reduz o tempo de espera e melhora o fluxo de veículos, permitindo uma experiência mais rápida e sem fricções.
Segurança: O uso da API para leitura de placas minimiza erros humanos e fraudes, garantindo que somente veículos autorizados utilizem o estacionamento.
Comodidade: Usuários não precisam se preocupar com tickets ou pagamentos manuais, pois todo o processo é automatizado e vinculado à placa do veículo.
